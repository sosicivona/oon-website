## Implementation Rules

- Use UMMO for the dossier or subject.
- Use Oyagaa-Oomo Network or OON for the organization.
- OON is an interface layer, not the archive and not the source.
- The true archives remain external.
- Keep source material, translation, summary, analysis, hypothesis, and OON interpretation visually distinct.
- If source data is missing, use a clear pending placeholder instead of guessing.
- Do not use public copy that says or implies archive names, source links, dates, IDs, quotes, researchers, institutions, statistics, downloads, or source-status labels are verified unless the approved source documents explicitly support it.
