# Misuse & False Attribution

This document defines how the website should handle alleged misuse, imitation, visual resemblance, and false attribution cases.

## Core Rule

A visual resemblance or later identification of a symbol is not evidence of source-chain connection.

This rule is especially important for Edelweiss/symbol cases and any public controversy involving signs that resemble or are later described as UMMO/OOMO-related.

## Attribution Standard

Do not write:

```text
he used the UMMO sign
```

unless source evidence establishes knowing use.

Preferred wording:

```text
a sign resembling or identified by some sources as the UMMO sign
```

Other acceptable cautious wording:

- "a symbol some observers associate with the UMMO sign"
- "a sign visually resembling the UMMO sign"
- "a later identification made by some sources"
- "a resemblance that requires source-chain review"

## Required Distinctions

When discussing possible misuse or false attribution, separate:

1. Visual resemblance
2. Later identification by observers
3. Claimed intentional use
4. Evidence of knowing use
5. Source-chain connection
6. OON interpretation

Do not collapse these layers.

## Edelweiss / Symbol Case Handling

For Edelweiss/symbol cases, use this baseline:

- A resemblance may be noted.
- A later identification may be described.
- Knowing use must not be claimed without source evidence.
- Source-chain connection must not be implied by visual similarity alone.
- OON interpretation must be labeled as interpretation.

## Misuse Language

Use "Misuse & False Attribution" as the section title.

This area should help readers understand how symbols, names, fragments, and claims can be reused, misread, copied, falsely attributed, or detached from source context.

It should not become a blame page or a sensational controversy page.

## Evidence Requirements

Before asserting misuse, the site should identify:

- what object or phrase was used
- who used it, if verified
- where it appeared
- whether use was knowing or only inferred
- what source connects it to the dossier, if any
- whether the identification is later commentary
- what remains uncertain

If these are not available, use neutral language and mark the item for review.

## Forbidden Shortcuts

Avoid:

- treating resemblance as proof
- implying motive without evidence
- implying source-chain connection from visual similarity
- using controversy to create urgency
- presenting OON interpretation as a source claim
- using screenshots without source context
